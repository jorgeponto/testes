Elementos label que não estão visíveis ou estão posicionados incorretamente.
O caso do input checkbox

1. Versão original com input type checkbox label

<input> <div> <label>

nota: esta versão parece natural para uma checkbox, embora tenha elementos entre o <input> e a <label>.

2. Construção com a input type checkbox label

<input> <label>

nota: posição natural de uma checkbox em que não há elementos entre <input><label>

3. Construção label input type checkbox - inversão da versão original

<label> <div> <input>

nota: esta versão é a menos natural para uma checkbox. Tal como a original também contém elementos entre a <label> e o <input>

4. Construção label input type checkbox - inversão da versão original, mas sem elementos entre eles

<label> <input>

nota: esta versão é menos natural para uma checkbox. Ao contrário da original, esta construção não contém elementos entre a <label> e o <input>
